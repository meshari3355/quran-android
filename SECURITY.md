# Security Policy

إذا وجدت مشكلة أمنية في التطبيق، لا تفتح Issue عام يحتوي تفاصيل استغلال مباشرة.

افتح تواصل خاص مع مالك المستودع أو أرسل وصفًا مختصرًا بدون أسرار أو بيانات مستخدمين، ثم شارك التفاصيل الكاملة بعد الاتفاق على قناة آمنة.

## معلومات لا يجب نشرها

- مفاتيح API.
- ملفات توقيع Android.
- بيانات حسابات Apple أو Google.
- بيانات مستخدمين حقيقية.
